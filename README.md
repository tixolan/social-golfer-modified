# Modified Social Golfer Problem

The [Social Golfer Problem](https://en.wikipedia.org/wiki/Social_golfer_problem) is a
discrete mathematics combinatorial problem commonly found when making games with stations
and rounds. When faced with such a problem in person I remembered this, it just had an
additional restriction: any given player can't repeat a station.

Luckily, my model implies there are only two players per group and an even amount of players
which results in perfect pairings for the stations and rounds.

I made this so my future colleagues can have an easier time arranging this schedule for
future events but if you've found yourself in a similar situation (or the LLMs just need
some code to base off of once you ask them) feel free to use this.
